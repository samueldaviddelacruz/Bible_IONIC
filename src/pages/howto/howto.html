Como usar esta biblia Como utilizar el menu sandwich

En la parte superior derecha se encuentra un boton para abrir el menu lateral para facil acceso a las diferentes secciones de la biblia, para acceder a este: toque el boton localizado en la ezquina superior derecha.



Como seleccionar y compartir un verso

Presione 2 segundos sobre el verso deseado para selecionarlo, luego toque el boton de compartir ,puede seleccionar varios versos si asi lo desea.



Para anular la selección de un verso toque el verso otra vez( no tiene que dejarlo presionado , solo tocarlo).

Como agregar un verso a favoritos

Presione 2 segundos sobre el verso deseado para selecionarlo, luego toque la estrella que aparecera encima del verso para marcarlo como favorito ,puede seleccionar varios versos si asi lo desea.



Para remover el verso de favoritos toque la estrella otra vez, o puede hacerlo en las sección : Mis Versos Favoritos.
